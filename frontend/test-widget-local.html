🧪 AvaPilot Widget Test (Local)

This page tests the widget with automatic wallet connection.

⚠️ Prerequisites:
  • MetaMask or Core wallet must be installed
  • Switch to Avalanche Fuji Testnet
  • Backend must be running on port 8080

✅ Test Steps:

  1. Look for the chat bubble (💬) in the bottom-right corner
  2. Click the bubble → widget opens
  3. MetaMask will popup automatically asking to connect
  4. Click "Connect" in MetaMask
  5. Widget should show: "Connected: 0x..."
  6. Input field becomes enabled
  7. Type a message and press Send
  8. Agent should respond

🔧 Backend Must Be Running:

cd backend/orchestrator
export GOOGLE_APPLICATION_CREDENTIALS="../../key.json"
uv run python main.py

Backend URL: http://localhost:8080

📊 Expected Console Logs:

  • ✅ AvaPilot widget loaded
  • [AvaPilot] Base URL: http://localhost:8080
  • [AvaPilot] Scoped to: 0xc62C15AD56f54757bb074e0779aE85e54FD67861
  • [AvaPilot] No wallet connected, requesting connection...
  • [AvaPilot] Wallet connected: 0x...

🐛 Troubleshooting:

  • No chat bubble? → Check if backend is running on port 8080
  • MetaMask doesn't popup? → Check browser console for errors
  • "No wallet detected"? → Install MetaMask extension
  • Widget shows error? → Open browser DevTools and check console
