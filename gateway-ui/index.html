How it works

→

AI Agent

Discovers + calls it

Lazy discovery

AI gets 45 clean tools, not 500. It searches services on-demand and calls specific contract functions. Scales to thousands of protocols.

Proxy-aware

Auto-detects proxy contracts and fetches implementation ABIs. USDC shows 55 callable functions, not 5.

Keys stay local

Private keys never leave your machine. All transactions signed locally. Fully open source — verify it yourself.

3 safety modes

Read — no wallet needed. Trade — send, swap, approve. Full — deploy contracts. You choose your risk level.

Add your protocol

Open a PR on GitHub adding your contract to seed.py. Once merged, every user who updates gets your protocol instantly.

For AI agent users

01

Install

pip install avapilot
avapilot seed
02

Connect to your AI

# Claude Desktop — add to claude_desktop_config.json
{
  "mcpServers": {
    "avapilot": {
      "command": "avapilot",
      "args": ["gateway", "--mode", "read"]
    }
  }
}

# OpenClaw — add to config
mcp:
  servers:
    avapilot:
      command: avapilot
      args: [gateway, --mode, trade]
03

Talk to Avalanche

"What DeFi protocols are available?"       → searches services
"Get me a swap quote for 10 AVAX to USDC" → calls Trader Joe
"What's my sAVAX staking balance?"         → reads sAVAX contract
"Swap 1 AVAX for USDC on Trader Joe"      → executes trade
04

Stay updated

# Get new services + features
pip install --upgrade avapilot
avapilot seed

Architecture

┌─────────────────────────────────────────────────┐
│  AI Agent (Claude, GPT, etc.)                   │
│                                                 │
│  "What can I do on Avalanche?"                  │
│        ↓                                        │
│  search_services() → 12 protocols found         │
│  service_functions("Trader Joe") → 24 functions │
│  call_service("Trader Joe", "getAmountsOut")    │
│        ↓                                        │
│  "1 AVAX = 9.50 USDC on Trader Joe"            │
└─────────────────────────────────────────────────┘
        ↕  MCP Protocol (stdio)
┌─────────────────────────────────────────────────┐
│  AvaPilot Gateway                               │
│                                                 │
│  45 built-in tools (network, gas, contracts)    │
│  + lazy discovery (search → inspect → call)     │
│  + 12 registered services (374 functions)       │
│  + proxy-aware ABI resolution                   │
│  + local key signing (keys never leave machine) │
└─────────────────────────────────────────────────┘
        ↕  JSON-RPC
┌─────────────────────────────────────────────────┐
│  Avalanche C-Chain / P-Chain / L1s              │
└─────────────────────────────────────────────────┘
